## Setting up the Environment 
### Conda
1. Clone the repo.
```bash 
conda env create -f environment.yaml
``` 
2. Activate the environment using `conda activate ENVNAME` (which is taenv by default).


### pip / virtualenv
```bash
pip install -r requirements.txt
```
